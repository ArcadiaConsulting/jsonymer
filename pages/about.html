jsonymer == ( json + polymer )

jsonymer is a simple polymer element that makes editing json data in a web browser quick and painless.

Features

  • Edit Javascript Object key->value pairs
  • Add/Remove Items, as well as edit a property type (string/object/array)
  • Basic Tab support for super fast editing
  • Simple API, including plenty of events to eavesdrop on
  • Automatic Unique Key Generation
  • Lets you know if there is an issue with your data (duplicate keys, for instance)
  • Ability to add custom errors
  • Of course, built as a polymer web component for ease of use and integration
